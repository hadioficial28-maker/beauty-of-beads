# ✨ Beauty of Beads - Premium Handcrafted Jewelry E-commerce

A modern, elegant, and fully responsive e-commerce website for **Beauty of Beads** — a handmade jewelry and bead accessories brand.

![Beauty of Beads](https://images.unsplash.com/photo-1611591437281-460bfbe1220a?w=1200&q=80)

## 🌐 Live Demo

[Visit Website](https://beauty-of-beads.vercel.app) *(Deploy to Vercel to get your live URL)*

## 🎨 Design Features

- **Luxury & Feminine Aesthetic**
- **Color Palette:** Rose Gold, Soft Pink, White, Cream, Gold accents
- **Typography:** Playfair Display (serif) + Inter (sans-serif)
- **Smooth Animations** with Framer Motion
- **Mobile-First Responsive Design**
- **Dark/Light Mode Support**

## 🚀 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Theme:** next-themes

## 📦 Features

- ✅ Full e-commerce experience (Shop, Cart, Checkout, Wishlist)
- ✅ Product filtering, sorting, and search
- ✅ Dark/Light mode toggle
- ✅ Persistent cart & wishlist (localStorage)
- ✅ SEO optimized with schema markup
- ✅ Responsive across all devices
- ✅ Instagram gallery integration
- ✅ Customer reviews section
- ✅ Newsletter subscription
- ✅ Contact form & FAQ page
- ✅ Custom order support
- ✅ Product image zoom
- ✅ Social sharing

## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/beauty-of-beads.git
cd beauty-of-beads

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
